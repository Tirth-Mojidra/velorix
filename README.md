VELORIX

Velorix is a comprehensive browser-based cybersecurity workspace designed for anti-phishing analysis. It equips users with essential tools to inspect suspicious URLs, analyze raw email headers, and scan attachments safely.

🔗 **[Live Demo: Try Velorix Here](https://tirth-mojidra.github.io/velorix/)**

---

## ✨ Key Features

* **🔗 Phishing URL Detector**
  Analyze suspicious links for risky protocols, misleading domains, brand impersonation, and phishing keywords.
* **📧 Email Header Analyzer**
  Verify SPF, DKIM, DMARC, sender alignment, reply-to mismatches, and routing hop signals directly from raw email headers.
* **📁 Attachment Scanner**
  Inspect file metadata, risky extensions, double-extension tricks, macros, scripts, archives, and embedded URL hints without executing the files.
